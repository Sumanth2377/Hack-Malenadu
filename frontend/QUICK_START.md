# CareSyn AI - Quick Start Guide

## 🚀 Access the Application

After starting the dev server, navigate to:

### Main Routes:

1. **Dashboard** - `http://localhost:3000/dashboard`
   - System overview with stats
   - Recent alerts
   - Device status
   - Activity timeline

2. **Workflow Builder** - `http://localhost:3000/workflow`
   - Visual pipeline editor
   - 4-stage automation:
     - Trigger (Risk Detection)
     - Call Engineer
     - Send Report  
     - System Logs

3. **Devices** - `http://localhost:3000/devices`
   - Equipment inventory
   - Health monitoring
   - Real-time metrics

4. **Alerts** - `http://localhost:3000/alerts`
   - Alert management
   - Filter by status
   - Engineer assignment

5. **Logs** - `http://localhost:3000/logs`
   - System audit trail
   - Search & filter
   - Event details

## 🎯 Try the Workflow Builder

1. Go to `/workflow`
2. Select a **Risk Level** (Low/Medium/High/Critical)
3. Choose a **Machine ID** from dropdown
4. Select an **Engineer** to call
5. Choose **Send Via** method (SMS/Email/Notification)
6. Click **"Execute Workflow"**
7. Watch the pipeline animate through each stage!

## 🎨 Features to Explore

- **Hover effects** on all cards
- **Color-coded risk levels** (green/yellow/orange/red)
- **Real-time status indicators** (pulsing dots)
- **Smooth animations** between workflow stages
- **Responsive design** - try resizing your browser
- **Dark sidebar** with active state highlighting
- **Search functionality** in header and logs page

## 📱 Mobile View

Resize your browser window to see responsive layouts:
- Sidebar auto-collapses on mobile
- Grid layouts adapt to screen size
- Cards stack vertically on small screens

Enjoy exploring CareSyn AI! 🎉
